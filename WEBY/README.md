
# Moje webová stránka (O mně)

Otevřete soubor `index.html` v prohlížeči.

Upravte obsah stránky takto:
- V `index.html` nahraďte text "Vaše Jméno" svým jménem.
- Doplňte text v sekcích "O mně", "Dovednosti" a "Zkušenosti".
- Pokud chcete vlastní avatar, nahrajte obrázek `avatar.svg` nebo `avatar.png` do stejné složky.

Součásti:
- `index.html` – hlavní stránka (osobní profil)
- `styles.css` – styly a rozvržení
- `script.js` – interakce (menu, formulář)
- `avatar.svg` – zástupný avatar

Nasazení: soubory stačí nahrát na statický hosting (GitHub Pages, Netlify apod.).
